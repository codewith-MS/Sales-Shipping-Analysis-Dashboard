# Sales-Shipping-Analysis-Dashboard
This project presents an end-to-end Sales and Shipping Analysis Dashboard built using Power BI. It focuses on uncovering key business insights related to revenue, profitability, customer behavior, and shipping efficiency.


## Case Study: Sales and Shipping Performance Analysis
Problem Statement :The business is experiencing growth in sales but lacks clarity on profitability, cost efficiency, and operational performance.
Key questions include:
-> Which categories and markets are truly profitable?
-> Are high sales translating into high profit?
-> How efficient are different shipping modes?

Approach
To address these questions, the dataset was analyzed using Power BI with the following steps:
### Data Cleaning and Preparation
- Removed inconsistencies and ensured correct data types
- Created relationships between orders, customers, and shipping data
### Data Modeling
- Designed a structured data model for efficient analysis
- Created calculated measures using DAX (Profit Margin, AOV)
### Visualization
Built two dashboards:
- Sales Analysis
- Shipping Analysis
Used bar charts, line charts, and scatter plots for comparison and trend analysis


## Key Insights
- Standard Class shipping contributes the highest cost across markets
- Technology category drives the highest profit contribution
- Some orders show high sales but low or negative profit, indicating cost or discount issues
- Europe and Asia Pacific are major contributors to overall sales
- Same Day shipping is least used but relatively cost-efficient



## Outcome
This dashboard enables stakeholders to:
- Track key performance metrics in real time
- Identify inefficiencies in sales and shipping
- Make data-driven decisions to improve profitability and operations
